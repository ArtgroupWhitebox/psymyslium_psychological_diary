## Технологии

- Дизайн прототипа разработан в Figma 
- Код написан на нативном JS, CSS, HTML
- С использованием Local Storage 
- Вёрстка сайта адаптивная, с применением Grid и Flex

Это мое первое веб приложение, первый в жизни код
